## User Setup Required
After merging, developers need to run:
```bash
uv sync --dev
uv run pre-commit install
```
This installs the pre-commit hooks locally. The hooks only affect the pre-commit stage (not pre-push), preserving the existing `scripts/pre-push` symlink.
