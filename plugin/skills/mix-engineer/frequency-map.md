# Frequency Map: Instruments & Problem Areas

Reference when making EQ decisions. Each instrument's frequency ranges, character descriptions, common problems, and typical EQ moves.

## Kick Drum

| Range | Character | Common Issues |
|-------|-----------|---------------|
| 40-60 Hz | Sub, weight | Rumble if room is untreated |
| 60-100 Hz | Fundamental, punch | Masking with bass guitar -- one must yield |
| 100-250 Hz | Body, boominess | Cut here to tighten -- almost always too much |
| 2-5 kHz | Beater attack, click | Boost for definition, harsh if over-boosted |

## Snare Drum

| Range | Character | Common Issues |
|-------|-----------|---------------|
| 150-250 Hz | Body, weight | Thin if missing, boomy if excessive |
| 400-800 Hz | Boxiness | Almost always cut here |
| 1-3 kHz | Crack, snap | The defining characteristic -- boost for presence |
| 5-10 kHz | Sizzle, wires | Bottom mic captures most of this |

## Toms

| Range | Character | Common Issues |
|-------|-----------|---------------|
| 80-200 Hz | Fundamental (varies by size) | Floor tom lower, rack tom higher |
| 3-5 kHz | Attack, stick definition | Boost for cut-through |
| Variable | Ring/resonance | Find the ringing frequency (solo, sweep narrow cut), notch it |

## Overheads / Cymbals

| Range | Character | Common Issues |
|-------|-----------|---------------|
| 200-500 Hz | Body, warmth | HPF at 300-500 Hz for "cymbal only" sound |
| 2-6 kHz | Presence, definition | Harshness from crash cymbals |
| 8-16 kHz | Air, shimmer | Too much = ice picks, too little = dull |

## Hi-Hat

| Range | Character | Common Issues |
|-------|-----------|---------------|
| 300-600 Hz | Fundamental, body | Usually HPF away |
| 6-10 kHz | Sizzle, brightness | Can dominate a mix if too loud |
| 10-16 kHz | Air | Natural rolloff is fine |

## Bass Guitar

| Range | Character | Common Issues |
|-------|-----------|---------------|
| 40-80 Hz | Sub fundamental | Keep mono below 120 Hz |
| 80-200 Hz | Body, warmth | Fights with kick at 60-100 Hz |
| 200-500 Hz | Mud zone | Almost always cut -- the single most impactful bass EQ move |
| 700 Hz-1 kHz | Growl, presence | Boost for definition in dense mixes |
| 2-4 kHz | String noise, fret buzz | Cut if distracting, leave if you want articulation |

## Electric Guitar

| Range | Character | Common Issues |
|-------|-----------|---------------|
| Below 80 Hz | Rumble | HPF here on every guitar -- no useful content |
| 100-250 Hz | Body, warmth | Can be too thick, especially with multiple guitars |
| 250-500 Hz | Mud, boxiness | Cut to clean up, especially rhythm guitars |
| 2-4 kHz | Presence, bite | Fights with vocals -- the critical complementary EQ zone |
| 4-6 kHz | Edge, aggression | Too much = fatiguing |
| 8-10 kHz+ | Air, noise | LPF here for rock (nothing useful above for distorted guitars) |

## Acoustic Guitar

| Range | Character | Common Issues |
|-------|-----------|---------------|
| 100-250 Hz | Body, fullness | Proximity effect if close-miked -- HPF at 80-120 Hz |
| 250-500 Hz | Boxiness | Cut 2-3 dB for clarity |
| 2-5 kHz | Clarity, pick attack | The sparkle zone -- boost gently |
| 5-10 kHz | Sparkle, air | Brightens the instrument but also string noise |

## Vocals

| Range | Character | Common Issues |
|-------|-----------|---------------|
| Below 200 Hz | Proximity rumble | HPF at 80-120 Hz on every vocal (higher for thin voices) |
| 200-500 Hz | Body, chest | Too much = thick/muddy, too little = thin |
| 800 Hz-1.5 kHz | Nasal, honk | Narrow cut if nasal quality is objectionable |
| 2-5 kHz | Presence -- the money zone | This is where vocals cut through a mix. Boost here, cut guitars here. |
| 5-10 kHz | Sibilance, brilliance | De-esser territory. Dynamic EQ better than static for sibilance. |
| 10-16 kHz | Air, breathiness | Gentle shelf boost for "air" -- adds intimacy and expensive sheen |

## Piano / Keys

| Range | Character | Common Issues |
|-------|-----------|---------------|
| Below 60 Hz | Sub rumble | HPF -- nothing useful down here on most keyboards |
| 100-300 Hz | Body, warmth | Can mask bass guitar |
| 300-500 Hz | Mud | Cut for clarity in dense mixes |
| 2-5 kHz | Clarity, attack | Boost for definition, fights with vocals |
| 5-10 kHz | Brilliance | Sparkle on acoustic piano, can be harsh on synth patches |

## Synths

| Range | Character | Common Issues |
|-------|-----------|---------------|
| 20-60 Hz | Sub bass | Keep mono, careful not to mask kick |
| 100-300 Hz | Warmth | Can thicken a mix or muddy it -- depends on arrangement density |
| 300 Hz-1 kHz | Body | Mid-heavy synths fight guitars for space |
| 2-5 kHz | Presence, edge | Can mask vocals -- complementary EQ needed |
| 5-10 kHz | Brightness | High-frequency content on supersaw/pad types can eat headroom |

## Common Multi-Stem EQ Conflicts

| Pair | Conflict Zone | Solution |
|------|--------------|---------|
| Kick vs Bass | 60-100 Hz | Complementary EQ: one gets the sub, the other gets the punch |
| Bass vs Guitars | 100-250 Hz | HPF guitars at 80-120 Hz, cut bass mud at 200-400 Hz |
| Guitars vs Vocals | 2-4 kHz | Cut guitars at vocal presence frequency -- the defining EQ move |
| Keys vs Guitars | 300 Hz-1 kHz | Pan separation + complementary midrange EQ |
| Multiple Vocals | 1-5 kHz | Slight frequency offset per voice, different reverb sends |
