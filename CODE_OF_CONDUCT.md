# Code of Conduct

This project follows the **Contributor Covenant Code of Conduct, version 2.1**.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Instances of unacceptable behavior may be reported to the project team at **phantom-audio@proton.me**.

All complaints will be reviewed and investigated promptly and fairly. The project team is obligated to maintain confidentiality with regard to the reporter of an incident.
